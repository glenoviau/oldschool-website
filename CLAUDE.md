# Project: Claude Website Build

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript (no frameworks)
- Hosted on GitHub Pages
- Contact form via Web3Forms, access_key value = "ff3a8846-1c9f-4be5-bd92-53b417784282" 
- Be sure to use the Web3Forms hCaptcha option for form submission


## Code Style
- Use CSS custom properties (variables) for all colours and fonts
- Mobile-first responsive design — always write mobile styles first
- No inline styles — all styling goes in style.css
- Use semantic HTML elements (nav, main, article, section, footer)
- JavaScript: ES modules only, no jQuery, no libraries unless discussed first

## File Structure
- index.html — homepage
- style.css — all styles
- script.js — all JavaScript
- /images — all image assets (use WebP format where possible)
- /pages — additional HTML pages

## Design Constraints
- Avoid Inter and Roboto fonts — only use Highway Gothic font
- Colour palette: #A8CEE0 (light blue), #e94560 (chiffon cream), #000000 (Black)

## Important Rules
- Never modify the contact form HTML without asking first
- Always test mobile layout before considering a task done
- Keep JavaScript accessible — all interactive elements must work with keyboard
- Do not add npm dependencies without discussing first

## Asset Rules
- Never generate, recreate, or approximate image assets in CSS or SVG
- All images are pre-prepared in /Design Input reference them directly
- When given a mockup image, use it for layout reference only
- Do not modify image dimensions, cropping, or styling unless explicitly asked
